Comment pouvons-nous vous aider ?

Compte

Pour créer un compte, cliquez sur "Créer" dans la barre de navigation ou cliquer sur le bouton "Commencer" de notre page d'accueil. Suivez les instructions pour entrer le détail de votre évènement.

La sécurité de vos informations est notre priorité. Nous utilisons un cryptage avancé pour protéger vos informations personnelles et vos détails de paiement. Vos informations ne seront jamais partagées avec des tiers sans votre permission.

Problèmes du site web

Si vous rencontrez des problèmes de connexion à votre compte, assurez-vous d'abord que votre nom d'utilisateur et votre mot de passe sont corrects. Si le problème persiste, veuillez contacter notre support client.

Si notre site web ne s'affiche pas correctement, essayez de rafraîchir la page ou d'effacer le cache de votre navigateur. Si le problème persiste, vérifiez si votre navigateur est à jour. Notre site est optimisé pour les dernières versions de Chrome, Firefox, Safari et Edge.
